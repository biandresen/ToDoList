#### Project tools: 
* <a href="https://github.com/users/biandresen/projects/1">GitHub Projects</a>

## About the project
### Responsive website application with features including:
* Create tasks
   * Parameters:
     * Title name
     * Note
     * Project name
     * Due Date
     * Color
     * Priority
     * Check/Uncheck
* Edit tasks
* Delete tasks
* Filter tasks
  * All tasks
  * Today
  * Tomorrow
  * This month
* Filter after project names
* Light/Dark mode
* Local storage of tasks and color mode
